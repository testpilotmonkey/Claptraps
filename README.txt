Welcome to Claptraps!

Controls
--------

Escape - switch between game and editor

Controls for game:
Arrow keys - move
Space - action button (cut down bushes)
Enter - Status screen

Controls for editor:
Left click - place item
Shift/space - change item to place
W/A/S/D - move map around (if bigger than screen)
Ctrl - place Dave
M - set minimum score
N - new map (press escape to cancel)
= - save map

You may have to turn off your browser's "Search as you type" feature to use all of the buttons above
